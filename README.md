# 注意事项

- 由于框架并非运行在浏览器中，所以 JavaScript 在 web 中的一些能力都无法使用，如 document、window 等对象
- 不支持直接的 dom 操作,很多开源 web 组件不能直接移植
- [节点查询](https://open-doc.dingtalk.com/microapp/dev/selector-query)
